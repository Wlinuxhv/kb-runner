## 修复说明

### 1. 临时目录路径修复
在 `cmd/kb-runner/main.go` 中，将 `tempDir` 转换为绝对路径：
```go
tempDir := cfg.Execution.TempDir
if !filepath.IsAbs(tempDir) {
    tempDir = filepath.Join(execDir, tempDir)
}
```

### 2. 脚本路径修复
在 `internal/adapter/adapter.go` 中：
- 添加了临时目录创建检查
- 将结果文件路径转换为绝对路径
- 将脚本路径转换为绝对路径

### 3. Python API修复
在 `scripts/python/kb_api.py` 中添加了函数包装器，使脚本可以直接调用 `kb_init()`, `kb_save()` 等函数。

### 4. 脚本修复
- 添加了 `scripts/bash/network_check.sh`
- 添加了 `scripts/python/performance_check.py`
- 修复了 Python 脚本在 exec 模式下的执行问题
