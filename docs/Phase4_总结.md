### 2. Bug 修复

在测试过程中发现并修复了以下问题：

#### Bug 1: YAML 解析包含注释
**问题**: 配置解析时 weight 值包含了行内注释
```bash
# 错误
Weight: 0.3#占 30 分

# 修复后
Weight: 0.3
```

**修复**: 改进 `_kb_load_step_config` 函数，使用 `sed 's/#.*//'` 移除注释

#### Bug 2: jq JSON 对象传递
**问题**: 多行 JSON 字符串直接传递给 jq 导致解析失败
```bash
# 错误
jq ".steps += [$step_json]" file.json

# 修复后
jq --argjson step "$step_json" '.steps += [$step]' file.json
```

**修复**: 使用 `--argjson` 参数正确传递 JSON 对象
