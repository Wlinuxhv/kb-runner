## 核心功能

### 1. 配置驱动得分

每个 KB 脚本配置文件 `case.yaml`：
```yaml
scoring:
  max_score: 100.0
  steps:
    - name: "步骤 1"
      weight: 0.3  # 占 30 分
      expected_status: "success"
```

### 2. 得分计算规则

- 步骤状态 = 期望状态 → 获得权重分
- 步骤状态 ≠ 期望状态 → 得 0 分
- 最终得分 = (实际得分 / 总权重) × 100

### 3. 执行方式

```bash
# 单个执行
./kbscript/run.sh KB 名称

# 批量执行
./kbscript/run_all.sh

# CLI 执行
./bin/kb-runner run --all
./bin/kb-runner run --category hardware
./bin/kb-runner run --kb KB1,KB2,KB3
```

### 4. Offline 模式

```bash
# 自动预处理日志包
./bin/kb-runner run --Q Q2026031700281 --all

# 使用已解压日志
./bin/kb-runner run --Q Q2026031700281 --unpreprocesslog \
  --processed-path /path/to/logs --all
```

### 5. 结果文件

```
~/kb-runner/workspace/results/
└── Q2026031700281/
    ├── KB1_exec-123_result.json      # 单个 KB 结果
    ├── KB2_exec-123_result.json
    └── ranked_results_exec-123.json  # 排名总览
```
